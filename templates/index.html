As STEM students we have gone through many different formulas which are supposed to have linear relation, of the top of my head, best example is Ohm's law, it states that Voltage and Current have a linear relation with each other but when we conduct the experiment we don't really observe a perfectly linear relationship, the values attained are somehwat linear and reason is because while conducting experiment there is variation in heat which causes a variation of resistance too. In these situations, we were taught to just connect the first point and last point and finish up the experiment but it does not give the most accurate equation of line, to get the most accurate line, we need to use a statistical method called linear regression, the line obtained from this is called the line of best fit.
Let us call the x values as independent variable and y the dependent variable.
It is assumed that the two variables are linearly related, so our task is find the function that predicts y values as accurately as possible. That is what this calculator does.
As the task is to find the line that best fits, we use priciple of least squares to find that.
